## Executive Summary

This proposal outlines a two-session short course for PhD students, postdocs, and faculty to learn how Claude Code can support real economics research workflows. The structure is grounded in the Claude Code Full Course and the economist-focused Compound Engineering training materials developed through AI MBA.

---

## Why This Training Now

Agentic coding tools like Codex and Claude Code are reshaping intellectual work, including academic research. But most existing trainings are still too generic, too developer-centric, or too disconnected from the actual workflows economists care about. Dr. Aniket Panjwani is a Northwestern-trained economics Ph.D. who works daily with agentic coding tools and teaches economist-first workflows through AI MBA, YouTube, and private trainings.

This course is designed to bring researchers up to speed on the parts of agentic coding that are most useful in practice for economics research: data cleaning, replication work, analysis extension, code generation and debugging, and faster iteration on active projects.

---

## Program Story and Structure

This version is designed as a compact two-day short course. Each live session is paired with selected chapter segments from the free 3-hour Claude Code Full Course, so the live time can focus on questions, demonstrations, and participant research problems rather than basic setup.

The default orientation is Claude Code plus the Compound Engineering workflow, which is currently the most natural entry point for economists. If useful, the course can also be adapted toward Codex, though it is usually better to choose one primary tool orientation in advance.

### Session 1: Research Attack with Compound Engineering (90 min)

- Set up the core workflow for economists: tools, terminal basics, Wispr Flow, permissions, and safety
- Introduce the Compound Engineering loop: brainstorm, plan, work, review, compound
- Live "research attack" on realistic economics tasks such as data cleaning, replication work, and empirical analysis extension

**Pre-watch before Session 1**

- Full course chapters 3:46–15:09 and 23:51–49:33
- A Practitioner's Guide to Compound Engineering

### Session 2: Skills, MCPs, and Subagents (90 min)

- Convert one-off prompting into reusable research workflows through custom skills
- Show how MCPs and subagents extend Claude Code for administrative automation, research assistance, and review loops
- Run a second live research attack centered on participant problems and workflow customization

**Pre-watch before Session 2**

- Full course chapters 49:33–1:18:53, 1:18:53–1:25:52, 1:25:52–1:42:31, and 1:50:35–2:01:56

## Before the Course: Required Software Setup

These installation steps are also walked through in the free 3-hour Claude Code Full Course, but they are listed here explicitly so participants know what should be ready before the first live session.

- **Wispr Flow** for dictation and higher-context interaction
- **Claude Code** as the default tool for the workshop
  - Windows participants should first install WSL, then install Claude Code inside it
- **Compound Engineering plugin** inside Claude Code
- **Git**
- **GitHub account and GitHub CLI** (gh)
- **VS Code**
- **Codex app** as an optional secondary tool if the audience wants some discussion oriented that way
  - macOS and Windows participants should install the Codex app
  - Linux participants can use Codex on the web for now and join the Linux app waitlist

Full source material: https://www.youtube.com/watch?v=HR1lI4V0oKE
